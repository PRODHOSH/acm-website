About the Event

What is Server Surfers?

Server Surfers is a full-day immersive cybersecurity experience hosted by the ACM Student Chapter at VIT Chennai. Featuring a keynote from an industry expert, a live KIPS threat simulation, and a nail-biting Capture The Flag competition.

Whether you're a seasoned security researcher or just beginning your journey into ethical hacking, Server Surfers is designed to challenge, educate, and inspire the next generation of cybersecurity professionals.
